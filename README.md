# 🏦 CashKeep: Cosmic Vault Edition
**CashKeep** is a high-performance, minimalist financial tracking application made to help users track their expenditure on a daily basis
Live site link
https://moseee6.github.io/CashKeep/
---

 Features
* **Real-Time Liquidity Metrics:** Instantly see your net balance, total inflows, and total outflows.
* **Authorization Portal:** Securely input transactions with categorized labeling (Salary, Business, Lifestyle, etc.).
* **Audit History:** A detailed, historical log of every entry into the vault.
* **Local Storage Sync:** Your data never leaves your machine. Everything is stored locally in the browser's data node.
* **Cosmic UI:** Fully responsive design .
